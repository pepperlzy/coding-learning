text-decoration 它是以下四个属性的简写 text-decoration-line ：文本修饰的位置，如下划线underline，删除线line-through text-decoration-color：文本修饰的颜色 text-decoration-style：文本修饰的样式，如波浪线wavy实线solid虚线dashed text-decoration-thickness：文本修饰线的粗细
